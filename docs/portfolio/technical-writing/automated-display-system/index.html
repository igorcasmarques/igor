User manual of the automated display system for museums

Download the full PDF

This user manual is the capstone project of Algonquin College's Technical Writer Certificate Program. I made it with Ato Dankwa and Polina Gankina.

Summary​

This is the user manual of an automated display system that was designed for the Ingenium museums in Ottawa by students in the Electrical Engineering Technology program at Algonquin College.

The purpose of this user manual is to show Ingenium team members how to use the automated display system to measure and control the lighting of museum items. Tailoring the lighting to the characteristics of an item on display has the following benefits:

  • Better viewing experience for the museum visitors
  • Prevention of light-related damage to the item

This user manual has five parts:

  1. Introduction of the user manual
  2. Descriptions of the automated display system’s case and web app
  3. Setting up the automated display system, including powering up and automating
  4. Handling the automated display system, including adjusting by hand and maintaining
  5. Reference material
